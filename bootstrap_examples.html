Text/Typography

Default Font

Default font-size is 14px, with a line-height of 20px. This is applied to the <body> element and all paragraphs (<p>).

<h1> - <h6>

Example :

h1 heading (36px)

h2 heading (30px)

h3 heading (24px)

h4 heading (18px)

h5 heading (14px)
h6 heading (12px)

Markup :


  <h1>h1 heading (36px)</h1>

  <h2>h2 heading (30px)</h2>

  <h3>h3 heading (24px)</h3>

  <h4>h4 heading (18px)</h4>

  <h5>h5 heading (14px)</h5>

  <h6>h6 heading (12px)</h6>

Alerts

Alerts are created with the "alert" class, followed by one of the four contextual classes "alert-success", "alert-info", "alert-warning" or "alert-danger"

Example :

× Success! Indicates a successful or positive action.
× Success! Indicates a successful or positive action.

Markup :

<div class="alert alert-success">

    <a href="#" class="close" data-dismiss="alert" aria-label="close">&times;</a>

  <strong>Success!</strong> Indicates a successful or positive action.

</div>

Buttons

Buttons are created with the "btn" class,followed by one of the four contextual classes "alert-success", "alert-info", "alert-warning" or "alert-danger". The classes that define the different sizes are: "btn-lg", "btn-md", "btn-sm", "btn-xs"

Example :

Markup :

<button type="button" class="btn btn-default">Default</button>

<button type="button" class="btn btn-primary">Primary</button>

<button type="button" class="btn btn-success">Success</button>

<button type="button" class="btn btn-info ">Info</button>

<button type="button" class="btn btn-info btn-sm">Info</button>

<button type="button" class="btn btn-info btn-xs ">Info</button>

Tooltip

The Tooltip plugin is small pop-up box that appears when the user moves the mouse pointer over an element.

Example :

Hover over me

Markup :

    <a href="#" data-toggle="tooltip" title="Hovered!">Hover over me</a>

Popover

The Popover plugin is similar to tooltips; it is a pop-up box that appears when the user clicks on an element. The difference is that the popover can contain much more content.

Example :

Toggle popover

Markup :

    <a href="javascript:void(0)" data-toggle="popover" title="Popover Header" data-content="Some content inside the popover">Toggle popover</a>

Grid System

The grid system allows up to 12 columns across the page. The grid system is responsive, and the columns will re-arrange automatically depending on the screen size.

Basic Structure

 <div class="row">

  <div class="col-d*-s*"></div>

  <div class="col-d*-s*"></div>

  <div class="col-d*-s*"></div>

</div>

Note:  (d*)-> device specific class [xs,sm,md,lg] , (s*)-> size of the column [1-12]. 1.create a row ( <div class="row">). 2.add the desired number of columns (tags with appropriate .col-d*-s* classes). 3.The numbers in (s*) should always add up to 12 for each row.

device specific classes (d*)

The classes below can be combined to create more dynamic and flexible layouts.

  • xs (for phones <767px)
  • sm (for tablets >767px)
  • md (for desktops >992px)
  • lg (for larger desktops >1200px)

Equal Columns

Note: For this example the columns are styled(height and background) to make it clear. The text like "1 0f 2" can be replaced by another content
